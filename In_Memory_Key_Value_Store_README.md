# In-Memory Key-Value Store

A Redis-inspired in-memory key-value database built using C++17.

## Features
- CRUD Operations
- Key Expiration (TTL)
- Thread-Safe Operations
- Concurrent Request Handling
- Memory Optimization

## Tech Stack
- C++17
- STL
- Multithreading
- Mutexes
- Unordered Maps
- Object-Oriented Programming

## Supported Commands

### SET
```bash
SET user:1 "Paresh"
```

### GET
```bash
GET user:1
```

### DELETE
```bash
DELETE user:1
```

## Concepts Demonstrated
- Object-Oriented Programming
- STL Containers
- Memory Management
- Multithreading
- Concurrency Control
- System Design
